**步骤4：**  
检查情节比例和公式整合。  

**情感公式：**  
- **快乐：**  
  强度 = (事件满意度 / 期望值) × 情感系数。  
- **悬念：**  
  强度 = (紧张程度 / 好奇心触发) × 情感系数。  
- **恐惧：**  
  强度 = (威胁级别 × 1/应对能力) × 情感系数。  
- **愤怒：**  
  强度 = (感知冒犯 × 权利侵犯) × 情感系数。  
- **悲伤：**  
  强度 = (损失大小 × 1/情感韧性) × 情感系数。  

**核心情感机制：**  
- **情感 = 压力累积 + 冲突升级 + 宣泄释放。**  
- **满意度 = 压力累积 + 冲突升级 + 宣泄释放。**  
- **微小满意度 = 小目标达成 + 即时奖励 + 轻微情感波动。**  
- **满意度预期 = 已知信息 + 模糊提示 + 延迟满足。**  
- **小满意度 = 局部冲突 + 短期胜利。**  
- **大满意度 = 长期铺垫 + 压力峰值 + 最终胜利。**  
- **压力进展 = 初始压力（开场） + 中层压力（冲突/转折）。**  
- **情感波动 = 上升压力 + 小满意度缓冲 + 大满意度释放。**  
- **信息差距 = 已知信息 (A) + 未知信息 (B) + 不对称驱动的冲突/悬念/转折 (C)。**  
- **新颖性 = 类型惯例 – 颠覆 + 逻辑合理性。**  
- **冲突公式 = 目标 + 障碍 + 行动 + 后果。**  
- **期待公式 = 渴望 + 希望 + 挑战 + 延迟满足。**  

**步骤5：**  
分析相关内容和当前章节情节，通过情感计算层优化（公式细化）并提供参考指数。  

**动态参数校准：**  
**E = (S × C) / (R^α + β) × γ**  
- **S：** 基础刺激（事件强度）  
- **C：** 角色特质系数（性格匹配）  
- **R：** 抵抗因子（成长曲线函数）  
- **α/β：** 类型调整变量  
- **γ：** 章节节奏权重（铺垫/高潮差异）
